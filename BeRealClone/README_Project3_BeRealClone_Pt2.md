# Project 3 - BeRealClone (Pt. 2)

Submitted by: **David Castaneda**

**BeRealClone (Pt. 2)** is an app that adds BeReal-style behavior to the existing Parse-backed feed: users can upload a post (photo + optional caption), view a feed limited to recent posts, unlock other users’ posts only after posting themselves (with a blur gate), and interact with posts via comments that display the commenter’s username. Posts also include time and (when available from photo metadata) location. **[TODO: tweak this description if you want]**
